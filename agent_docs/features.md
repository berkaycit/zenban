# Features

## Kanban Board

- 3 fixed columns: To Do, In Progress, Done
- Multiple boards with sidebar navigation
- Drag-and-drop cards between columns

## Card Management

- Create cards with Cmd+N or toolbar button
- Edit cards by double-clicking
- Delete cards via hover X button or edit sheet
- Cards show title with 3-line limit

## Board Management

- Create boards with Cmd+Shift+N
- Rename boards via context menu
- Delete boards via context menu or swipe

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Cmd+N | New card |
| Cmd+Shift+N | New board |

## Data Storage

- JSON file at `~/Library/Application Support/com.berkaycit.zenban/boards.json`
- Auto-saves with 500ms debounce
- Loads on app launch
